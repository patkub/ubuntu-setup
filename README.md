# ubuntu-setup

### Environment

Tested on the following distributions
  - Ubuntu 24.04.2 LTS (Noble Numbat)
  - Kubuntu 24.04.2 LTS (Noble Numbat)

### Install

Make the script executable
```bash
chmod +x ./ubuntu_install.sh
```

Run the script and select 1 to install.
```bash
./ubuntu_install.sh
```
